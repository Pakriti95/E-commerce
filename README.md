# ShopNest - Full Stack E-Commerce Platform

A full-stack E-Commerce web application built using the MERN Stack (MongoDB, Express.js, React.js, Node.js).

## Features

- User Registration & Login
- JWT Authentication
- Product Browsing
- Shopping Cart
- Checkout System
- Order History

## Tech Stack

### Frontend
- React.js
- Redux Toolkit
- CSS

### Backend
- Node.js
- Express.js

### Database
- MongoDB
